## 第2章 C语言概述
主要讲了C程序的主要结构，函数是C程序的基本模块。函数（function）由函数头，和函数体构成。
```c
#include<stdio.h>
int main(void)        //函数头
{                     //函数体
	int num;          //声明
	num = 1;          //语句
	printf("my favorite number is %d because it is first number.\n",num);
	return 0;
}                          
```
其他一些关键词：
- 函数 fuction
- 变量 variable
- 声明 declaration
- 语句 statement
- 预处理 prepocessing
- 头文件 header
- 关键字 keyword 关键字是C语言的词汇
- 标识符 identifier
